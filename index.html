UTM5: Личный кабинет

Для операторов связи

Основные изменения относительно штатного лк:

  • Новое оформление (изготавливается по заказу)
  • Интуитивно понятный интерфейс, основанный на статистике поведения пользователей
  • Детальный баланс, аб. плата, об. платеж, кредит
  • Умная работа с обещанным платежом
  • Показ нестандартной информации об абоненте
  • Фильтрация и автокоррекция данных абонента
  • Проверка валидности номеров телефонов, адресов, и.т.д
  • Автологин по IP
  • Смена мак-адресов в учетной записи, с проверкой дубликатов
  • Умная смена тарифных планов
  • Предложение о смене старых тп, предложение о смене тп исходя из потребления трафика
  • Удобные графики
  • Система кэширования данных из биллинга

Безопастность:

  • Полностью переписаный код личного кабинета, без использования фреймворков.
  • Глубокая проверка вводных данных
  • Щадящий, регулируемый режим работы с биллингом.
  • Защита от бутфорса
  • Защита от ботов
  • Защита от бутфорса с разных ип (ботнет)
  • Защита от подмен кук, подмен ип
  • Доступ по ИП
  • Работает на свежих версиях PHP, apache
  • Syslog

Требования:

  • UTM5 5.2-008 и новее.
  • Модуль URFA
  • Apache + php5.6+
